# 적립식 투자(DCA) 수익률 시뮬레이터

> 매일, 매주, 매월 꾸준히 주식을 사면 얼마나 벌 수 있을까?

## 목적

주식 투자에서 **적립식 투자(DCA, Dollar-Cost Averaging)**란 한 번에 큰 돈을 넣는 대신, **일정한 주기로 조금씩 꾸준히 사는 방법**입니다.

예를 들어 매달 월급날마다 QLD(나스닥 2배 레버리지 ETF)를 1주씩 산다고 하면, 주가가 비쌀 때도 사고 쌀 때도 사게 되어 **평균 매수 단가가 자연스럽게 낮아지는 효과**가 있습니다.

이 앱은 **"내가 이 주식을 이 기간 동안 꾸준히 샀다면 지금 수익이 얼마일까?"**를 쉽게 확인할 수 있도록 만든 시뮬레이터입니다.

## 기능

| 기능 | 설명 |
|------|------|
| 기간 설정 | 시작일과 종료일을 자유롭게 지정할 수 있습니다 |
| 매수 주기 선택 | 매일 / 매주(월요일) / 격주(월요일) / 매월(첫 영업일) 중 선택 |
| 티커 변경 | QLD뿐 아니라 TQQQ, SPY, AAPL 등 원하는 종목으로 변경 가능 |
| 투자금 vs 평가금 그래프 | 내가 넣은 돈과 현재 가치가 어떻게 벌어지는지 한눈에 확인 |
| 수익률 추이 그래프 | 기간 중 수익률이 어떻게 변했는지 확인 |
| 주가 차트 + 매수 시점 표시 | 언제 샀는지, 평균 매수단가는 얼마인지 시각적으로 확인 |
| 적립식 vs 일시불 비교 | 같은 돈을 첫날에 한꺼번에 넣었을 때와 비교하여 복리 효과 확인 |
| 상세 매매 데이터 | 매수일별 매수가, 누적 주수, 수익금 등 상세 내역 제공 |

## 활용

### 이런 분들에게 유용합니다

- **주식 초보자**: "적립식 투자가 정말 효과가 있을까?" 궁금할 때
- **ETF 투자자**: QLD, TQQQ 같은 레버리지 ETF의 적립식 투자 효과를 확인하고 싶을 때
- **투자 전략 비교**: 매일 vs 매주 vs 매월 중 어떤 주기가 나에게 맞는지 비교할 때
- **일시불 vs 적립식**: 목돈을 한 번에 넣는 것과 나눠서 넣는 것 중 뭐가 유리한지 알고 싶을 때

### 실행 방법

```bash
# 1. 필요한 패키지 설치
pip install -r requirements.txt

# 2. 앱 실행
streamlit run app.py
```

브라우저에서 `http://localhost:8501`로 접속하면 바로 사용할 수 있습니다.

### 사용 예시

1. 왼쪽 사이드바에서 **티커**(예: QLD)를 입력합니다
2. **시작일/종료일**로 시뮬레이션 기간을 정합니다
3. **매수 주기**를 선택합니다 (매일, 매주, 격주, 매월)
4. **회당 매수 주수**를 설정합니다 (기본 1주)
5. 자동으로 계산된 수익률과 그래프를 확인합니다
